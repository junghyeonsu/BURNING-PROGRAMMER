C++
C 컴파일러(Cpre)에 클래스, 상속 클래스, 스트롱 타이핑, 인라인 확장, 기본 인수,가상 함수, 함수 이름, 연산자 오버로딩, 참조, 제약조건, type-safe free-store 메모리 할당 (new/delete), 개선된 자료형 검사, BCPL 스타일의 1줄 코멘트(//)를 포함한 여러 기능들을 추가하였다.
템플릿이 안전하고 API를 일반화하는 데 적합하기 때문에 단순하다. 즉, 시간과 성능을 통제할 수 있다. 

isocpp.org/

IDE로는 Visual Studio Code를 사용한다.
https://code.visualstudio.com/

속도가 빨라 민감한 인공지능 프로젝트에 유용하다. OOP 원칙도 가지고 있어 AI프로그램에 적합하다. 따라서 머신러닝, 딥러닝 라이브러리의 상당 부분은 C/C++로 작성되어 있다. API를 제공하며, 다른 프로그래밍 언어에 래퍼(wrapper)를 제공한다. 
C++를 사용하는 대표적인 응용프로그램에는 마이크로소프트 윈도우, 맥 OS, 어도비 포토샵, 마야 3D 소프트웨어, CAD, Mozilla Firefox 등이 있다.
